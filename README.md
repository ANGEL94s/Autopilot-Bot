Quick start (local)
1. Copy `.env.example` to `.env` and fill in:
   - TOKEN: bot token from @BotFather
   - OWNER_ID: your Telegram user id
2. Create virtualenv and install:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   python3 -m pip install -r requirements.txt
